-= SHORT INTRODUCTION =-
This library can be used, when it's done, to easily create a prompt on the Serial connection of an Arduino.
It will also include the ability to store/load configuration to/from EEPROM.

If you have ideas for improvement, please let me know @ chris (a) bleq nl.
